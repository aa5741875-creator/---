# Проєкт з бізнес-прогнозування

## Технічне завдання (ТЗ)
* **Мета проєкту:** Побудова прогнозу бізнес-показників (наприклад, обсягу продажів, попиту або прибутку) для прийняття управлінських рішень.
* **Вхідні дані:** Історичні бізнес-метрики (часові ряди або панельні дані).
* **Очікуваний результат:** Модель прогнозування з оціненою точністю та аналітичний звіт.

## Статус виконання задач
Актуальний трекінг задач ведеться у вкладці **Projects** цього репозиторію.
