# Проєкт: Прогноз товарообігу мережі на 18 місяців наперед

## 1. Технічне завдання (ТЗ)
* **Мета проєкту:** Створення моделі середньострокового прогнозування щомісячного товарообігу торговельної мережі з горизонтом планування 18 місяців для оптимізації закупівель та фінансового планування.
* **Об'єкт прогнозування:** Сумарний товарообіг мережі (або у розрізі ключових точок/категорій).
* **Вхідні дані (Датасет):** Історичні дані про продажі (часовий ряд), показники сезонності, маркетингова активність, святкові дні.
* **Горизонт прогнозу:** 18 місяців наперед (з розбивкою по місяцях).
* **Очікуваний результат:** Навчена прогностична модель з оціненою похибкою та фінальний звіт.
